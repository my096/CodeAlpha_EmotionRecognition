# Emotion Recognition from Speech

## Objective
Recognize human emotions from speech audio using Deep Learning.

## Dataset
Use:
- RAVDESS
- TESS
- EMO-DB

## Features
- MFCC Extraction
- CNN Model
- Emotion Prediction
- Streamlit Web App

## Run

pip install -r requirements.txt

python train_model.py

streamlit run app.py
